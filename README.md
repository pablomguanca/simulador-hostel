# 🏄 Surfer Hostel - Simulador de Habitaciones

¡Bienvenido a **Surfer Hostel**! Esta es una aplicación web interactiva diseñada para ayudar a los viajeros a encontrar su habitación ideal mediante un simulador inteligente basado en sus preferencias.

## 🔗 Demo en vivo
Podés ver el proyecto funcionando y probar el simulador acá:  
[https://pablomguanca.github.io/simulador-hostel/](https://pablomguanca.github.io/simulador-hostel/)

## ✨ Características principales
- **Simulador Inteligente:** Responde 3 preguntas clave y obtén el "match" perfecto con la habitación que mejor se adapta a vos.
- **Historial de Búsquedas:** Panel lateral interactivo que guarda tus últimas consultas localmente utilizando `localStorage`.
- **Galería Deslizable:** Carrusel de imágenes nativo con soporte para *scroll-snap* y controles intuitivos.
- **Diseño "Deep Ocean":** Interfaz moderna y sofisticada con una paleta de colores inmersiva, construida con arquitectura SASS.
- **Feedback Visual:** Notificaciones elegantes y modales de confirmación personalizados utilizando la librería SweetAlert2.

## 🛠️ Tecnologías utilizadas
Este proyecto fue construido utilizando:
- **HTML5:** Estructura semántica.
- **SASS:** Arquitectura modular de estilos utilizando metodología **BEM** y variables dinámicas.
- **JavaScript:** Lógica de filtrado, manejo del DOM, almacenamiento local y renderizado dinámico de tarjetas.
- **SweetAlert2:** Para una experiencia de usuario fluida en alertas y confirmaciones.
- **Google Fonts:** Tipografías *Playfair Display* y *DM Sans* para una jerarquía visual óptima.
